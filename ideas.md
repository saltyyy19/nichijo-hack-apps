# 日常ハックアプリのアイデア（軍資金なし、ブラウザ完結）

## コンセプト
サーバー代などのランニングコストをかけず（無料で公開できるGitHub Pagesなどを利用）、ブラウザの機能（LocalStorageなど）を最大限活用することで**軍資金0円**で始められるニッチな便利ツールです。

## アイデア3選

### 1. 底値（そこね）シンプルメモ
**誰のどんな面倒を解決する？**
家計簿アプリは入力が面倒。「いつも買う卵や牛乳が、どのスーパーでいくらだったか」だけを最速でメモ・比較したい主婦層や一人暮らし層向け。

**特徴（無料で作れる理由）**
*   **データ保存:** スマホのブラウザ内（LocalStorage）に保存するため、データベースサーバーが不要。
*   **機能:** 商品名と値段と店名を入れるだけ。一覧表示で安い順に並び替えられる。
*   **技術:** HTML + CSS + JavaScript のみ。

### 2. サブスク管理＆解約リマインダー
**誰のどんな面倒を解決する？**
「無料体験で登録したけど解約を忘れて課金されてしまった」「今月いくらサブスクに払っているか把握していない」という人向け。

**特徴（無料で作れる理由）**
*   **データ保存:** LocalStorage。
*   **機能:** サービス名、月額、更新日を入力。更新日が近づくと画面上で赤く警告を出す。（ブラウザの通知機能を使うことも可能）
*   **技術:** HTML + CSS + JavaScript のみ。計算と日付処理が中心。

### 3. フリマ出品用「コピペでOK」な紹介文メーカー
**誰のどんな面倒を解決する？**
メルカリやヤフオクなどで不用品を売る際、商品説明を考えるのが面倒な人向け。

**特徴（無料で作れる理由）**
*   **データ保存:** なし（その場でテキストを生成するだけ）。
*   **機能:** 「服」「家電」「本」などのカテゴリを選び、「ブランド名」「状態（傷あり等）」「使用期間」などのプルダウンを選ぶだけで、丁寧で検索されやすい紹介文テンプレートが完成。最後に「コピーする」ボタンを押すだけ。
*   **技術:** HTML + CSS + JavaScript のみ。テキストの組み立てロジックが中心。
